### defer

- defer表达式中变量的值在defer表达式被定义时就已经明确

```go
func a() {
    i := 0
    defer fmt.Println(i)
    i++
    return
}

执行结果：0
```

- defer表达式的调用顺序是按照先进后出的方式

```go
func b() {
    defer fmt.Print(1)
    defer fmt.Print(2)
    defer fmt.Print(3)
    defer fmt.Print(4)
}

执行结果：4321
```

- defer表达式中可以修改函数中的命名返回值

```go
func c() (i int) {
    defer func() { i++ }()
    return 1
}			
执行结果：2

func f() (r int) {
     t := 5
     defer func() {
       t = t + 5
     }()
     return t
}
执行结果：5
```
